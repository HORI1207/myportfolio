SSTWC

【目的】
シンプルな掲示板アプリです。CRUD機能の実装をしていく中でPHPとSQLの知識を深めていくために作成したポートフォリオです。

【使用したツールと技術】
Linux / VScode / docker / heroku / GitHub / HTML / CSS / php / MySQL,

【このアプリの作成で苦労した点】
前回のアプリ作成時にdockerがかなり不安定で苦労したので、今回はXAMMPを使用して作成しました。しかし、XAMMP環境で作成したアプリケーションのHerokuへのデプロイの方法についてわかりやすく載っている情報が見当たらなかったので、結局作成したファイルをdocker環境に移動してデプロイしました。 そのことが原因かは定かではないが多数のエラーが発生し、トラブルの解決にとても苦労しました。

【今後の課題、目標】
1,PHPとSQLをもっと深く理解して使いこなす。
2,完全オリジナルのアプリを企画して作りきる。
3,dockerコンテナを自作する。
4,AWSを使う。
